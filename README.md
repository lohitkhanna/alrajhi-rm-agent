# Al Rajhi Bank – RM Agent (Gemini Edition — FREE)

Powered by Google Gemini 2.0 Flash — completely free, no credit card needed.

## 🚀 Deploy to Vercel

1. Upload all files to GitHub (see previous guide)
2. Go to vercel.com → Import repo → Deploy
3. Share URL with team

## 🔑 Get Free Gemini API Key

1. Go to https://aistudio.google.com/app/apikey
2. Sign in with Google account
3. Click "Create API Key" — it's free
4. Paste into the app on first visit

## 📁 Files
- index.html, vite.config.js, package.json
- src/App.jsx — full RM agent
- src/main.jsx — entry point
